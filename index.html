Tambah content_type: "product" — ini wajib untuk catalog matching di Meta, langsung affect EMQ score yang jadi tujuan utama PRD ini.

event_id harus pakai checkout session ID yang unik. Tanpa ini, deduplication antara FE dan BE selama parallel run tidak akan bekerja dan data bisa double count.

external_id (hashed user ID Toco) tambahkan kalau user sudah login — salah satu signal EMQ tertinggi di CAPI.
